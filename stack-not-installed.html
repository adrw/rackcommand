Teamspeak Stack

Teamspeak is the world's best voice communication software for gamers. It allows you to speak on a chat channel with fellow computer users, much like a telephone conference call.

This stack will install the TeamSpeak server itself and licensed to handle up to 32 users. Anymore than that and a non-profit license will need to be obtained.

  • Recommended Server Specs
  • 1x Core (1.5GHz+) CPU
  • 256MB+ RAM
  • 1GB Disk
